INTRO

Wearable Application which improves working efficiency

The modern workplace consists not only the office, but also many other functions which offer staffs better working experience. Our idea is to improve the efficiency of staffs in the modern workplace by giving functions allow users find others in a short time or able to locate themselves and meantime generate a record of the time allocation.

Navigation system

Allowing users to find available directions to key workplaces through a starred line to indicate users' transit route​

Time Management System

Illustrating to users how much time they spend at different workplaces​

Fatigue prevention system

Notifying users when they have remained in the same place for too long to prevent over-exertion fatigue​
